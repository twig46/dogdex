# DogDex Web
This is the updated web version of the DogDex app, please note 
 1. I have removed the ability to have dogs persist through sessions (otherwise you would have no reason to buy the [app](https://play.google.com/store/apps/details?id=com.tobyv.dogdex) only A$1.99 / ~$1.49)
 2. The web version is still primarily designed for phones/ mobile devices, if you are wanting to open this site on a computer I would suggest shrinking the horizontal axis of the window
